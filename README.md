# Mapping the e-petition ecosystem through Social Media: mobilization in the EU across issues and ideologies

This repository contains supplementary information and code related to the paper “Mapping the e‑petition ecosystem through social media: mobilization in the EU across issues and ideologies”, published in Journal of Quantitative Description: Digital Media, 6(2026), 1–33. DOI: 10.51685/jqd.2026.001.

The code in the R file allows the creation of the figures and tables of the Manuscript as well as some of the figures from the Supplement, where possible without violating GDPR regulations or X/Twitter's terms of service.
In accordance with EU regulations, all data has been anonymised and specific microdata has been voluntarily omitted to prevent any risk of identifiability.
